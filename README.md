# 🎭 Meme Generator Web App

A simple and beginner-friendly **Meme Generator Web Application** built using HTML, CSS, and JavaScript. This project allows users to upload an image and add customizable top and bottom text to create memes easily.

---

## 🚀 Features

* 📷 Upload your own image
* ✏️ Add **top and bottom text automatically positioned**
* 🖱️ Drag and adjust text position if needed
* 📱 **Mobile-friendly full-screen UI**
* 🎨 Clean and simple user interface
* ❌ Removed unnecessary UI elements (like extra bars/download clutter)
* 💾 Download your generated meme

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
